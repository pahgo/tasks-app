# Product Overview

## Core Concept

A collaborative task system with:

* Ownership (tasks belong to a user)
* Sharing (tasks can be shared with others)
* Flexible organization (topics)
* Priorities and status

## Features (MVP)

* Google login
* Create/edit/delete tasks
* Task status: todo, in_progress, done, archived
* Task priority: low, medium, high
* Topics (many-to-many)
* Share tasks (viewer/editor)
* Live filtering

## Core Rules

* Users can only see:

  * their own tasks
  * tasks shared with them

## Future Features

* Comments
* Notifications
* Saved filters
* Analytics
