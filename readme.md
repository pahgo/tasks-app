# Tasks App (Supabase + React)

This project is a collaborative task management app with:

* Google authentication
* Task priorities and status
* User-defined topics
* Task sharing with permissions
* Live filtering UX

## Tech Stack

* Frontend: React
* Backend: Supabase (PostgreSQL + Auth + Realtime)
* Hosting: Vercel

## Goal

Build a secure, fast, and collaborative task app with excellent UX.

## How to use with AI agents

This repository contains structured documentation in `/docs`.

An AI agent (e.g. Claude) should:

1. Read all docs in `/docs`
2. Follow architecture decisions strictly
3. Implement incrementally (auth → tasks → sharing → filters)
